# Privacy-Policy
Privacy Policy — Black System Bot

The Black System bot processes message content only when required to provide context-based assistance and support features Discord servers.

Data Usage:

* Message content is processed in real-time to trigger help and utility features.
* Messages are not permanently stored outside Discord.
* No message content is used for AI or machine learning training.
* No user data is sold or shared with third parties.

Server administrators can disable message-based features at any time.

By using the bot, users agree to this data usage policy.
